# Edirectinsure Challenge

## Usage

```
# clone repository
git clone https://github.com/soniacurcialeiro/edirectinsure.git

# change directory
cd edirectinsure

# install composer packages
composer install

# run migrations
php artisan migrate

# start app
php artisan serve
```

*Visit [http://localhost:8000](http://localhost:8000) to view the page.*
